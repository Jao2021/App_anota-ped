# Anota_ped
Este projeto tem como objetivo ser um aplicativo para anotar pedidos em restaurantes e feiras alem de fazer um registro do inventario de produtos, relatório de pedidos feitos, relatório de clientes e apesentar informações sobre o quanto foi ganho nas vendas.

O projeto foi criado utilizando a linguagem dart e flutter, foi aplicado tambem um banco de dados Postgresql que reside em outro repositório no github.

Repositório do banco de dados : https://github.com/Jao2021/Banco_anota-ped

## Objetivo
- Site modelo para um restaurante japones.
- Praticar programação front-end em aplicativos.
